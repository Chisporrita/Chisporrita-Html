Types of microprocessors

+ CISC (Complex Instruction Set Computer): The instructions are in a complex form. It means that a single instruction can contain many low-level instructions. Besides, we can say that a single instruction has multiple addressing modes. As there are many operations in single instruction they use very few registers.

+ RISC (Reduced Instruction Set Computer): In this, the instructions are quite simple, and hence, they execute quickly. Moreover, the instructions get complete in one clock cycle and also use a few addressing modes only. Besides, it makes use of multiple registers so that interaction with memory is less.

+ EPIC (Explicitly Parallel Instruction Computing): It allows the instructions to compute parallelly by making use of compilers. Moreover, the complex instructions also process in fewer clock frequencies. Furthermore, it encodes the instructions in 128-bit bundles. Where each bundle contains three instructions encoded in 41 bits each and a 5-bit template.
